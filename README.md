# Arduino Integrated Car System
Designed, assembled, and coded a small robot car to drive along a track while turning right, left, and around using Arduino.

https://github.com/user-attachments/assets/c3757b37-e308-423b-8d45-2012a37088b3

The code contains various turning and driving functions. Magnetic encoders attached to the wheels were used to calculate and track the angle of rotation of the car. Pictures will be attached
